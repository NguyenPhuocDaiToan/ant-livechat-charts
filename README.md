# The Ant Livechat Library for Kubernetes
Deploy the Ant Livechat application to Kubernetes

## TL;DR
```
$ helm repo add ant-livechat https://antbuddy.github.io/ant-livechat-charts/
$ helm repo update
```

## Before you begin
### Prerequisites
- [Kubernetes 1.17.4+](http://kubernetes.io/docs/getting-started-guides/)
- [Helm 3.4+](https://github.com/helm/helm#install)

### Add Repo
```
$ helm repo add ant-livechat https://antbuddy.github.io/ant-livechat-charts/
```
